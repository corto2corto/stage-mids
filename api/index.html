Jours anormaux

Modèles : Xt ~ Poisson(λNt) ou NB(μNt, r) — la moyenne scale avec le volume du jour Nt, μ est la fréquence moyenne du mot, et r règle la dispersion (r petit = série irrégulière ; r → ∞ redonne Poisson). Estimation par maximum de vraisemblance (forme fermée pour λ ; statsmodels pour μ, r). Les lois tracées sont les mélanges exacts sur les vrais Nt de la période. L'adéquation est testée par un χ² sur les résidus de Pearson (χ² = Σ (Xt − moyennet)²/variancet, ddl = jours − paramètres estimés) : p < 0,05 rejette la loi. Même méthode et mêmes sorties que les fiches PDF du mémoire.
